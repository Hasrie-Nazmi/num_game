# NUM_GAME
## Simple Python game that uses basic arithmetic such as addition, subtraction, multiplication, division to try and get the highest score before running out of lives.
## This game was developed using Python and Pygame
## Objective:
Get the highest score by before running out of lives or score reaches 0
